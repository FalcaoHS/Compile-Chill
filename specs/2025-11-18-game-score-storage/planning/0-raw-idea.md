# Raw Idea: Game Score Storage

## Feature Description

Create Prisma schema for game scores, API routes to save scores, and database models for users, games, and scores with proper relationships.

## Source

From Product Roadmap - Item #5

## Notes

This feature is foundational for tracking player performance across all games. It will enable:
- Storing individual game scores
- Linking scores to users and games
- Establishing relationships between User, Game, and Score models
- Creating API endpoints for score submission
